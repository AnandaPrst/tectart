# Deskripsi Project
Project ini merupakan aplikasi sistem penjualan toko sederhana berbasis website yang dibuat untuk memenuhi tugas project akhir TecArt. Aplikasi ini berjalan secara lokal dan digunakan untuk membantu proses:
- Pengelolaan data produk
- Transaksi penjualan (POS)
- Melihat laporan penjualan
- Menampilkan dashboard informasi

Aplikasi terdiri dari folder frontend, backend, dan docs sebagai tempat dokumentasi.
---

# Teknologi yang Digunakan
Project ini dikembangkan menggunakan teknologi berikut:
- HTML
- CSS
- JavaScript
- Git & Github (version control)
